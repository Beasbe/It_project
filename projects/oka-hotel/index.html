О проекте

Проект по модернизации телефонной связи в гостинице "Ока". Замена устаревшего оборудования на современную цифровую АТС.

**Работы выполненные:**

- Проектирование телефонной сети для гостиницы

- Монтаж АТС Panasonic на 200 номеров

- Подключение всех номеров

- Настройка дополнительных услуг

- Обучение персонала

**Результат:** Современная телефонная связь для гостиницы с расширенным функционалом.
